ALERT TYPE
Indicates the type of Alert

  • Unknown: Unknown alert type.
  • Issuer Alert: The alert came from the issuing bank.
  • Cancel: The transaction was cancelled before settlement.
  • Dispute: this is a customer dispute alert.
  • Fraud: this is a fraud alert.

INTERNAL ALERT ID
Unique identifier in Midigator database

CASE ID
Unique case identifier as provided by either Ethoca or Verifi

RECEIVED DATE
Date when the Alert was created by either Ethoca or Verifi.

TRANSACTION DATE
Date when the original transaction happened

AMOUNT
Amount of the original transaction

CURRENCY
Currency of the original transaction

BIN
First six digits of number of the credit card used in the original transaction

LAST 4
Last four numbers of the credit card used in the original transaction

FIRST NAME
Cardholder's first name

LAST NAME
Cardholder's last name

ARN
The Acquirer Reference Number associated with the transaction (if available)

REASON CODE
The reason provided for the chargeback

DESCRIPTOR
Descriptor associated to the MID through which the transaction was run

VENDOR NAME
This is the Alert Vendor that originated the Alert in the first place
